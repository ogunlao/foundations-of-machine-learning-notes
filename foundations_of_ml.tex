	\section{Regression Problem}
	
	Given a set of datapoints, $ D = \left\{\left(X_i, y_i\right)\right\}_{i=1}^n $ where $x_i$ are the features and $y_i$ are the target, corresponding to the features. \\
	The hypothesis function is \\
	\begin{equation}
		h_{\theta}(X) = \theta_0 + \theta_1x_1 + \theta_2x_2 + ... + \theta_nx_n
		= \sum_{i=0}^{n}\theta_ix_i \qquad (x_0=1)
	\end{equation}
